mgn-scroll-nav

Implement scroll position with linked navigation. If scroll position intrudes the specified content area, assign class="active".

https://github.com/frontend-isobar-jp/mgn-scroll-nav


new mgnScrollNav(
    ".j-scrollnav",
    {
        boxElm: '.f-box',
        activeElm: '.icon',
        posFix:  '50%',
        edgeJudge: false

    }
);
